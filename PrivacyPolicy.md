The app may request the following permissions in order to provide its features:

The app may request access to the device’s photo gallery so the user can select images to use within the game. Selected images are used only locally on the device and are not transmitted or shared.

These permissions are optional and are only used when the related feature is used.

Any images or audio created or selected within the app remain stored **only on the user’s device**. The developer does not receive or store this data.
